# 🎨 Shadestore – Centralized Paint Retail Management System

Welcome to **Shadestore** 🚀
A comprehensive **centralized paint retail management system** built using Django that integrates **inventory management, e-commerce, billing, painter services, field staff management, document generation, chat system, and product tracking** into a single platform.

---

## 📌 Project Overview

Shadestore is designed to streamline and digitalize the operations of paint retailers by combining **business management tools** with a **customer-facing platform**.

This system enables:

* Efficient inventory and stock management
* Seamless online shopping experience
* Service booking for painters
* Field staff coordination and tracking
* Automated billing and document generation
* Real-time communication and order tracking

---

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```bash id="7g3vpx"
git clone https://github.com/amalujose189/Shadestore-main_project-.git
```

### 2️⃣ Navigate to Folder

```bash id="r9m1zw"
cd Shadestore-main_project-
```

### 3️⃣ Create Virtual Environment

```bash id="3k8vnt"
python -m venv venv
```

### 4️⃣ Activate Environment

```bash id="y1p6dh"
venv\Scripts\Activate.ps1
```

### 5️⃣ Install Dependencies

```bash id="2b4qjs"
pip install -r requirements.txt
```

### 6️⃣ Apply Migrations

```bash id="c8u2ex"
python manage.py makemigrations
python manage.py migrate
```

### 7️⃣ Run Server

```bash id="r6z3qa"
python manage.py runserver
```

### 8️⃣ Open Browser

```id="p0w5kx"
http://127.0.0.1:8000/
```
